# Spec-Driven Development (App Saúde)

Siga rigorosamente estas etapas na ordem:

1. Revisar ou criar PRD.md
2. Criar ou atualizar Spec.md (Tech Spec)
3. Criar Implementation Plan (tarefas pequenas e ordenadas)
4. Implementar feature por feature com revisão
5. Atualizar database-schema.md quando necessário

Sempre pergunte ao usuário antes de avançar para a próxima etapa.
Nunca pule planejamento.
