# Regras para Dados de Saúde e Checklists

- Checklists são templates globais ou por clínica
- Execuções de checklists (shift_checklists) são imutáveis após conclusão
- Sempre registrar quem realizou a ação (`created_by` / `profile_id`)
- Mantenha histórico completo de registros
- Priorize clareza e usabilidade para usuários não-técnicos (cuidadores e administradores)
