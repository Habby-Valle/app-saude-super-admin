# Regras de UI/UX (Super Admin)

- Interface clean, profissional e com alto contraste
- Use tabelas com paginação e busca para listas grandes (clínicas, usuários, pacientes)
- Dashboard deve mostrar KPIs claros e gráficos simples
- Priorize legibilidade (fontes grandes, espaçamento generoso)
- Suporte completo a modo dark
- Mantenha navegação intuitiva e rápida
- Garanta responsividades em celulares, tablets e ect
